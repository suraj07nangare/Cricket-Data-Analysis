# Cricket-Data-Analysis
This notebook demonstrates the use of Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn for performing basic data analysis and visualization.
Project Title

Data Visualization using Matplotlib and Seaborn

Overview

This project focuses on exploring and visualizing data using Python libraries such as Matplotlib and Seaborn. The notebook demonstrates how to create different types of plots and customize them for better understanding of data.

Technologies Used:
Python
Pandas
NumPy
Matplotlib
Seaborn

Key Concepts Covered:

Line Plot creation and customization
Multiple line comparisons
Plot styling (color, markers, line styles)
Axis scaling and limits
Grid and legend usage
Scatter Plot visualization
Relationship analysis between variables

Datasets Used:
Custom dataset for basic plotting
Cricket dataset (Virat Kohli and Rohit Sharma performance)
Batter dataset for strike rate vs average analysis

Features:
Visualization of time-series data
Comparative analysis of multiple variables
Use of real-world datasets
Clean and structured plotting approach
How to Run

Install required libraries:

pip install pandas numpy matplotlib seaborn
Open the notebook in Jupyter Notebook or Google Colab
Ensure datasets (.csv files) are in the same directory
Run all cells sequentially
Output
Line graphs showing trends over time
Comparative plots of cricket player performance
Scatter plots showing relationships between variables
Use Case

This project is useful for beginners learning data visualization and serves as a foundation for data analysis and machine learning projects.
